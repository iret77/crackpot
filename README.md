# Die Logische Kugel

**Eine Reihe von Diskussionspapieren über Symmetrie, Artefakte und die Struktur der Realität**

*Christian Wendler — physikinteressierter Laie*

---

## Überblick

Diese Reihe untersucht eine zentrale Vermutung: Viele scheinbar fundamentale Eigenschaften der physikalischen Realität — Singularitäten, der Kollaps der Wellenfunktion, die Richtung der Zeit — könnten Artefakte unserer Beschreibung sein, nicht Eigenschaften der Natur selbst.

Das Fundament bilden drei Prinzipien:
- **Symmetriebruch ist nie gratis.** Jede reale Asymmetrie hat einen identifizierbaren Mechanismus.
- **Kein Sonderstatus.** Keine Beobachtungsebene hat privilegierten Zugang zur Realität.
- **Keine freien Parameter.** Wo eine Theorie eine Zahl braucht aber nicht erklärt, gibt sie zu, dass ihr etwas fehlt.
